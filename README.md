This project is an AI-Based Study Planner designed to help students plan their studies in a smart and organized way. The application creates personalized study schedules based on the subjects selected and the time available for studying.

The goal of this project is to improve time management, study consistency, and learning efficiency by using artificial intelligence to generate better study plans.

Students can add subjects, generate a study plan, and track their progress easily through a simple and user-friendly interface.
